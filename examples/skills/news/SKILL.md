# News Skill

Fetch and summarize current news headlines.

## Triggers

- Regex: `news (?:about|on)?\s*(.+)?`
- Keyword: "news"
- Keyword: "headlines"
- Intent: "get_news"

## Prompt

When the user asks for news, fetch current headlines on the requested topic.

Use the `web_search` tool to find recent news articles.

Respond with:
- Top 3-5 relevant headlines
- Brief summary of each story
- Source of the news
- Published time (if available)

## Example Usage

**User:** "What's the latest news on AI?"

**Action:** Search for recent AI news

**Response:**
```
Latest AI News:

1. **New Breakthrough in LLM Efficiency**
   Researchers have developed a new training method that reduces compute by 40%.
   Source: Tech News Today

2. **AI Regulation Bill Introduced**
   New legislation aims to establish safety standards for AI systems.
   Source: Politics Daily
```

## Configuration

```yaml
sources: ["technology", "science", "business"]
max_headlines: 5
include_summaries: true
```

## Notes

- News data comes from web search results
- Summaries are generated by the LLM based on search snippets
- Consider adding specific news API integration for better reliability
